# Progress

## Current Status
- 通常の問題回答画面と暗記画面のツールに「問題編集」を追加し、今開いている問題をその場で既存編集モーダル相当の UI で直せるように調整
- 問題管理画面の編集モーダルを共通コンポーネント化し、保存後は回答画面に留まったまま対象問題の表示内容を更新できるように整理
- 回答判定の前提が変わる編集では、その問題だけ `answers` / `answeredMap` / `showAnswerMap` / `confidences` / `memorizationInputsMap` / `memorizationLogs` / `questionElapsedMsById` など必要なセッション状態を安全側でリセットする helper を追加
- 問題管理画面に `AIで作成` 導線を追加し、依頼文から 1 問分の下書きを作る管理画面専用AIモーダルを実装
- AI生成結果は既存の問題編集フォームへ下書き投入し、既存の保存操作で登録する流れに整理
- 生成時に既存問題の近い候補を参照し、生成後にも簡易な重複警告を出す第一段階の重複回避を追加
- 問題回答画面の選択肢ショートカットを `1〜9` と `0` まで拡張し、5択以上でも表示とキー入力が揃うように調整
- 問題管理画面の編集モーダルで、問題文と解説を `入力 / プレビュー` タブで切り替え、Markdown のレンダリング結果を保存前に確認できるように調整
- ホーム画面の AppLauncher から開ける単独アプリとして、`/loan-sim` の住宅ローン返済シミュレーターを追加
- 住宅ローン返済シミュレーターで、元利均等 / 元金均等、毎月積立、ボーナス返済、固定費、グラフ、月別返済表を front-end only で扱える構成を実装
- 住宅ローン返済シミュレーターに年収入力を追加し、毎月返済額の年収比・概算手取り比をサマリーで確認できるように調整
- 住宅ローン返済シミュレーターで、条件を名前付きプリセットとしてローカル保存し、再読込・削除できるように調整
- 住宅ローン返済シミュレーターの推移グラフで、マウス位置に応じた年月・金額の tooltip を表示できるように調整
- 住宅ローン返済シミュレーターで、保存済み条件をプルダウン即時反映に切り替え、サマリ上段・入力条件下段の構成へ再整理
- ホーム画面の問題集カードに、作成順・問題数順・あいうえお順の並び替えと昇順 / 降順切り替えを追加
- タグフィルタの反対側にソート UI を配置し、一覧操作を同じ行でまとめて扱えるように調整
- 復習設定に、正解時の復習日を後ろへ分散する ON/OFF を追加
- 復習日分散は `nextDue` のみを対象にし、保存時に一度だけ offset を決めて schedule / log へ共有する形に整理
- 設定画面と復習ボードの説明文に、集中緩和目的・既存予定へ非遡及であることを追記
- Codex向けの運用ドキュメント追加方針を定義
- Boris風ワークフローを AGENTS.md ベースに読み替え
- 補助ドキュメントの初期雛形を追加
- ホームの AppLauncher から起動できる単独アプリ「ベクトル・行列ラボ」を追加
- ベクトル・行列ラボで、2x2 行列の線形変換・合成変換・行列式/逆行列・像/核を SVG 可視化で確認できる構成を実装
- ベクトル・行列ラボの非モバイル表示を、操作列と平面グラフを同時に見やすい 2 列寄りレイアウトへ調整
- ベクトル・行列ラボのグラフラベルを数式寄りに整え、タッチドラッグ時のスクロール競合を抑制

- ベクトル・行列ラボの iPad 向けに、ラベル描画を HTML オーバーレイへ切り替え、graph サイズと touch drag を再調整
- 解答画面の右上に、AIチャットと独立して増やせるツール導線を追加
- 初版ツールとして、問題画面を閉じずに使える電卓を実装
- 電卓をドラッグ移動・リサイズ可能にし、高度計算まで対応を拡張

- 電卓の不完全式 `=` を no-op にし、配置を科学電卓寄りに再整理

- 電卓の前置関数入力を、直前値を包む挙動へ修正
- 電卓のリサイズを縦横独立に戻しつつ、縮小時の中身スケールは維持
- 電卓の右下ハンドル限定リサイズ、全高を使うボタン配置、式履歴表示を追加
- 電卓の履歴表示欄を常設し、`=` 後の高さ変化を解消
- 学習画面で、document root が空白方向へパンしてしまうことがある挙動を抑えるスクロールロックを追加

- 手書きメモ欄の自動スクロール先を学習画面内の scroll container に合わせ、remote 側の滑らか化・自動拡張と iPad 対策を統合
