# Progress

## Current Status
- Codex向けの運用ドキュメント追加方針を定義
- Boris風ワークフローを AGENTS.md ベースに読み替え
- 補助ドキュメントの初期雛形を追加
- ホームの AppLauncher から起動できる単独アプリ「ベクトル・行列ラボ」を追加
- ベクトル・行列ラボで、2x2 行列の線形変換・合成変換・行列式/逆行列・像/核を SVG 可視化で確認できる構成を実装
- ベクトル・行列ラボの非モバイル表示を、操作列と平面グラフを同時に見やすい 2 列寄りレイアウトへ調整
- ベクトル・行列ラボのグラフラベルを数式寄りに整え、タッチドラッグ時のスクロール競合を抑制
- ベクトル・行列ラボのパラメータ入力ラベルも KaTeX 表示へそろえ、記号の読み取りを統一
- ベクトル・行列ラボの iPad 向けに、ラベル描画を HTML オーバーレイへ切り替え、graph サイズと touch drag を再調整
- 解答画面の右上に、AIチャットと独立して増やせるツール導線を追加
- 初版ツールとして、問題画面を閉じずに使える電卓を実装
- 電卓をドラッグ移動・リサイズ可能にし、高度計算まで対応を拡張
- 電卓の未閉鎖括弧の補助表示と、関数単位のバックスペース削除を追加
- 電卓の不完全式 `=` を no-op にし、配置を科学電卓寄りに再整理
- 電卓表示の 3 桁区切りと、ボタン群が収まる最小ウィンドウサイズを追加調整
- 電卓に `±` / `%` / `AC/C` を追加し、比率固定の縮小表示にも対応
- 電卓の前置関数入力を、直前値を包む挙動へ修正
- 電卓のリサイズを縦横独立に戻しつつ、縮小時の中身スケールは維持
- 電卓の右下ハンドル限定リサイズ、全高を使うボタン配置、式履歴表示を追加
- 電卓の履歴表示欄を常設し、`=` 後の高さ変化を解消
- 学習画面で、document root が空白方向へパンしてしまうことがある挙動を抑えるスクロールロックを追加
